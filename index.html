How to Use this App

  • 1
    Input Course Units
    You will have to input course units ranging from 0-5 for each course
  • 2
    Input Course Grade
    Input Course Grade which can be A, B, C, D, E or F. Note that Inputs should be in capital letter
  • 3
    Click on Calculate GPA
    Clicking on the Calculate GPA button will display your GPA for the selected course units and grades selected above

Fast, Accurate, Reliable and Trustworthy

GPA Calculator is a Web Application that helps Students to Calculate their Grade Point Average with ease. It serves both Students and Lecturers in Nigerian Tertiary Institutions(University and Polytechnic).
All Semester Results can be calculated in one click. All you need to do is to input the Grades and Course Units for each course, Select the GPA Scale(University or Polytechnic), Add more rows for more courses, and click on Calculate GPA.
